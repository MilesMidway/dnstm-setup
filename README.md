## 🔍 What is dnstm-setup?

dnstm-setup is an interactive wizard that guides you to set up **dnstm**. dnstm automates your network configuration using DNS technology. This tool helps set up DNS tunneling and VPN servers with simple steps. It cuts down the time and effort usually needed for such tasks.

You do not need technical skills to use dnstm-setup. The wizard walks you through on Windows computers with step-by-step instructions. It configures your system automatically once you provide a few required details.

Key points about dnstm-setup:
- Designed for Windows users
- Works with dnstm for DNS-based network setup
- Helps configure DNS servers and tunnels
- No programming or command-line needed
- Interactive and user-friendly

## ⚙️ Using the Setup Wizard

The setup wizard helps configure:

- DNS Server: Sets up a DNS server for your local network.
- DNS Tunnel: Creates secure DNS tunnels for VPN use.
- Network Settings: Adjusts your system to route traffic using dnstm.

The wizard asks you to enter:

- Hostname or IP address used for your DNS server
- Port numbers for connection
- Credentials or keys for secured tunnels
- Preferences for routing behavior

If you are unsure about any input, the wizard explains each option. You can also keep default values recommended for most cases.

## 🔧 Common Tasks During Setup

### Setting Hostname or IP

Provide the hostname or IP address of your DNS server. This is where your device sends DNS queries.

### Choosing Port Numbers

Enter ports used for DNS and tunneling traffic. The common default is port 53 for DNS.

### Configuring Credentials

If your network requires credentials or keys, the wizard prompts you to enter these. Make sure you have this info from your network administrator.

If you face issues during setup, try these steps:

- Check if your firewall allows dnstm-setup to work.
- Restart your computer and try the setup again.
- Verify that your internet connection is stable.

If errors show specific messages, write them down or take a screenshot. This helps in seeking assistance from community forums or support channels.
